# projeto-marketing-bancario-EDA
Este projeto consiste em uma Análise Exploratória de Dados (EDA) aplicada a uma campanha de marketing bancário realizada por um banco português, cujo objetivo é compreender quais fatores influenciam a adesão de clientes a um depósito a prazo promovido por meio de contatos telefônicos. Também envolve a criação de um Dashboard p/ insights..
